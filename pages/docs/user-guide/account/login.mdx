# Masuk

Berikut adalah cara untuk masuk ke dalam aplikasi SIPBUK:

1. Buka halaman utama [SIPBUK](/).
2. Klik tombol `Masuk` pada bagian kanan atas halaman.
3. Masukkan **Email** dan **Kata sandi** yang telah didaftarkan.
4. Klik tombol `Masuk`.

   <Callout type="info">
     Jika Email Belum Terverifikasi, silahkan melakukan [Aktivasi
     Akun](/docs/user-guide/account/account-activation) atau Jika anda Lupa Kata
     Sandi, silahkan melakukan [Pengaturan Ulang Kata
     Sandi](/docs/user-guide/account/forgot-password).
   </Callout>

5. Jika berhasil, maka anda akan diarahkan ke halaman **Dashboard** SIPBUK.
