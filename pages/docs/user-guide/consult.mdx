# Konsultasi

Pada bagian ini merupakan panduan dalam melakukan konsultasi dengan sistem. Silahkan ikuti langkah-langkah berikut ini.

1. Buka halaman utama [SIPBUK](/).
2. Klik tombol [`Konsultasi`](/consult) pada navigasi atas.
3. Klik tombol `Mulai` atau langsung geser ke bawah untuk melihat daftar pertanyaan.
4. Pilih pertanyaan yang sesuai dengan kebutuhan Anda dan Pilih jawaban `Sangat Tidak Yakin` jika pertanyaan tidak relevan dengan kebutuhan Anda.

   <Image
     className="shadow-lg rounded-lg mt-4 border border-gray-200"

   />

5. Setelah memilih jawaban, pada bagian bawah halaman klik tombol `Yakin Dan
Diagnosis` untuk melihat hasil diagnosis.
