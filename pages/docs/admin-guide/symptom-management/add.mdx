# Tambah Data Gejala

Pada bagian ini merupakan panduan dalam melakukan penambahan data gejala pada aplikasi SIPBUK. Berikut langkah-langkahnya:

1. Pada Halaman [Dashboard Admin](/admin) Klik tombol `Lihat` pada bagian bawah Kalimat **Total Data Gejala** atau klik halaman berikut [Data Gejala](/admin/symptoms).
2. Klik tombol `Tambah Data`.
3. Isi formulir yang tersedia seperti:

   - **Keterangan Gejala\***
   - **URL Gambar**

4. Klik tombol `Simpan` untuk menyimpan data gejala baru.
