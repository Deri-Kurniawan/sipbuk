# Ubah Data Gejala

Pada bagian ini merupakan panduan dalam melakukan perubahan data gejala pada aplikasi SIPBUK. Berikut langkah-langkahnya:

1. Pada Halaman [Dashboard Admin](/admin) Klik tombol `Lihat` pada bagian bawah Kalimat **Total Data Gejala** atau klik halaman berikut [Data Gejala](/admin/symptoms).
2. Klik tombol `Ubah Data` pada kolom data yang ingin diubah.
3. Ubah formulir yang tersedia seperti:

   - **Keterangan Gejala\***
   - **URL Gambar**

4. Klik tombol `Simpan` untuk menyimpan perubahan data gejala.
