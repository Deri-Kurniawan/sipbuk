# Pengelolaan Data Gejala

Panduan pengelolaan data gejala ini ditujukan bagi pengguna yang memiliki peran sebagai admin.

Untuk dapat mengelola data gejala, anda dapat mengakses halaman [Data Gejala](/admin/symptoms) terlebih dahulu.
