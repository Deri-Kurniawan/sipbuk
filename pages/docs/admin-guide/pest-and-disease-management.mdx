# Pengelolaan Data Hama dan Penyakit

Panduan pengelolaan data hama dan penyakit ini ditujukan bagi pengguna yang memiliki peran sebagai admin.

Untuk dapat mengelola data hama dan penyakit, anda dapat mengakses halaman [Data Hama dan Penyakit](/admin/pests-deseases) terlebih dahulu.
