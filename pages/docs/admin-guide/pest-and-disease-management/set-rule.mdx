# Atur Rule Data Hama atau Penyakit

Pada bagian ini merupakan panduan dalam melakukan pengaturan Rule data hama atau penyakit yang terikat gejala pada aplikasi SIPBUK. Berikut langkah-langkahnya:

1. Pada Halaman [Dashboard Admin](/admin) Klik tombol `Lihat` pada bagian bawah Kalimat **Total Data Hama dan Penyakit** atau klik halaman berikut [Data Hama dan Penyakit](/admin/pests-deseases).
2. Klik tombol `Atur Rule` pada kolom data yang ingin diatur.
3. Klik _Checkbox_ untuk memilih gejala yang terikat dengan hama atau penyakit yang ingin diatur. Klik _Dropdown_ untuk memilih nilai ketidakpastian pakar.
   <Image
     className="shadow-lg rounded-lg mt-4 border border-gray-200"
     src={setRulePestOrDesease}
     width={1080}
     height={720}
     alt="Gambar Pertanyaan ke 1 dengan Jawaban Yakin"
     priority
   />
4. Klik tombol `Simpan Rule` untuk menyimpan rule hama atau penyakit.
