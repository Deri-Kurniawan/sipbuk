# Tambah Data Hama atau Penyakit

Pada bagian ini merupakan panduan dalam melakukan penambahan data hama atau penyakit pada aplikasi SIPBUK. Berikut langkah-langkahnya:

1. Pada Halaman [Dashboard Admin](/admin) Klik tombol `Lihat` pada bagian bawah Kalimat **Total Data Hama dan Penyakit** atau klik halaman berikut [Data Hama dan Penyakit](/admin/pests-deseases).
2. Klik tombol `Tambah Data`.
3. Isi formulir yang tersedia seperti:

   - **Nama Hama atau Penyakit\*** (Mulai dengan kata Hama atau Penyakit)
   - **Solusi** (Cara mengatasi hama atau penyakit)
   - **Bahan Aktif** (Bahan aktif yang digunakan untuk mengatasi hama atau penyakit)

4. Klik tombol `Simpan` untuk menyimpan data hama atau penyakit baru.
5. Jika berhasil maka anda dapat melanjutkan untuk melakukan pengaturan _Rule_ seperti pada dokumentasi berikut poin 3 [Pengaturan Rule](/docs/admin-guide/pest-and-disease-management/set-rule)
